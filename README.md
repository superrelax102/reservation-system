# 予約管理システム

このプロジェクトは、Spring BootとJUnit、AWSを使用して構築する予約管理システムです。
実用性とテスト品質にこだわった、プロトタイプ開発を目指しています。
2026/4/11時点で開発中です。開発ステータスは適宜修正します。

## 技術スタック
- **Backend:** Java, Spring Boot, Thymeleaf
- **Test:** JUnit, Mockito
- **Database:** PostgreSQL (Docker)
- **Infrastructure:** AWS (App Runner, RDS)
